# Calculator Full Stack Application

## Overview
A Full Stack Calculator Application developed using React.js for the frontend and Spring Boot for the backend. The application performs basic arithmetic operations through REST APIs.

## Technologies Used

### Frontend
- React.js
- Axios
- HTML
- CSS
- JavaScript

### Backend
- Java 17
- Spring Boot
- REST API
- Maven

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Real Calculator UI
- Continue calculations using previous result
